Web Projects

Web App

This Web Application calculates various statistical information about a block of text. Namely this application will tell you the number of characters, number of words, number of lines, number of non-empty lines, the length of the longest line, the average word length, the ten longest words, and the ten most frequent words.
To use this application one writes text into the text box and then uses the compute stats to see their results at the bottom of the page. The clear button will clear the text box of all current text. The ex1 - ex3 buttons give some preset examples for the user.
